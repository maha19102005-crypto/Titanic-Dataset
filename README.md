# Titanic Survival Analysis Dashboard

## Overview

This project presents an interactive Power BI dashboard built using the Titanic dataset. It analyzes passenger demographics, survival rates, ticket class, age, gender, and fare to uncover factors that influenced survival.

## Features

* Interactive KPI cards for passenger and survival statistics.
* Visualizations of survival by gender, passenger class, and age.
* Slicers and filters for dynamic data exploration.
* Easy-to-understand charts for identifying survival patterns.

## Tools Used

* Power BI
* Power Query
* DAX
* Titanic Dataset

## Dashboard Insights

* Total passengers and survival rate
* Survival by gender
* Survival by passenger class
* Age distribution of passengers
* Fare analysis

## Conclusion

This dashboard demonstrates data cleaning, visualization, and business intelligence techniques using Power BI to analyze passenger survival patterns on the Titanic.
